------------------------------------------------------
Constant 		Meaning
------------------------------------------------------

------------------------------------------------------
ios_base::out 		Open file for writing.
------------------------------------------------------

------------------------------------------------------
ios_base::app 		Append to end-of-file.
------------------------------------------------------
ios_base::trunc 	Truncate file if it exists.
------------------------------------------------------
ios_base::binary 	Binary file.
